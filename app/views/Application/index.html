This is a demo Java app running on Heroku using the Play! framework. It uses the Force.com Web Service Connector (WSC) and the Partner jar to connect to a DE salesforce.com org. There are also versions of this app for Ruby on Rails, Python (Google App Engine) and Java (Google App Engine).

Run the demo

You can find the source for this project on github.
